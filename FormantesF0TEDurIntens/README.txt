Para rodar o script, é preciso uma segmentação e etiquetação prévia num textgrid.
Cada arquivo som/textgriddeve conter somente uma sentenca. Deve tambem conter uma linha de TextGrid para VV (de preferencia tier 1) e uma para numero de silabas (de preferencia tier 2).
O script foi criado para rodar no Praat. Nao requer a abertura de nenhuma janela, de modo que acredito seja simples de mudar para rodar no shell.
A versão 0.2 contem todos os procedimentos em separados, na forma de procedures, de modo que fica mais simples implementar os scripts na forma de modulos.
