This folder has template files, for rapid application development with the framework.

Guide step-by-step:

1) Run Eclipse and click in the main bar the options 'Window', and 'Preferences...';

2) Mark all templates and remove them clicking in 'Remove';

3) Import all new templates clicking on 'Import...';

4) After clicking on 'Import...', choose the directory ".../extras/templates/eclipse",
   and choose file 'template_simp.xml'.

All right! Now you may use the templates :-)
